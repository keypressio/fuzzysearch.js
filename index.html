A fuzzy search library for javascript.

FuzzySearch.js lets you filter an array by a string. It doesn't just match the string that's typed, but any string where the characters match anywhere.

For example, fzzjs would match FuzzySearch.js. Useful!

Try it now.

How to use?

Using FuzzySearch.js is simple. Use the following code:

var results = FuzzySearch(needle, haystack);

Where needle is what you're looking for, and haystack is an array of strings.

Everything is case insensitive, because, you know, it's fuzzy. And whitespace is trimmed.

You'll get back an array of objects. Each object will look like this:

{
    "item": "Final Fantasy VI: Sabin",
    "index": 0,
    "score": 940
}

After that, it's up to you. Filter the list, highlight matching characters. Whatever you want.
